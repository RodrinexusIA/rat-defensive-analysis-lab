\# RAT Defensive Analysis Lab



Defensive cybersecurity portfolio project focused on malware behavior analysis, detection engineering, and incident response.



\## Overview



This repository documents the defensive analysis of a Python-based Remote Access Trojan concept.



This public version is fully sanitized. It does not include working malware, payload generation, persistence, keylogging, screenshots, or remote command execution.
